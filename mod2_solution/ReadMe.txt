Module 2 Coding Assignment:

The page created here, provided 3 mockups: 
desktop, tablet, and mobile. 
It is implemented as one page (i.e. a responsive page).

A CSS file placed into a css folder, e.g., module2-solution/css. And no javascript framework is used.

The browser displays a desktop version of the site when the width of the browser window is 992px and above. 

Tablet view appears only if the width of the browser window is between 768px and 991px,

inclusively. Mobile view appears only if the width of the browser is equal to or less than 767px.
